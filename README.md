# Cold & Quiet Cryostat Model

STEP model of the custom cryostat used in the work [Cryogenic geometric anti-spring vibration isolation system](https://iopscience.iop.org/article/10.1088/1361-6501/ae5404/meta).

An online 3D render can be accessed through [this link](https://3dviewer.net/#model=https://raw.githubusercontent.com/mpa139/Cold_and_Quiet/refs/heads/main/cryo-design.step).
